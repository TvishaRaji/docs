This guide will walk you through the process of creating API keys for Anthropic using Rabbitholes. 

## Step 1: Access the Rabbithole Portal

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant1.png"
/>
- After logging in to [RabbitHoles.AI](http://RabbitHoles.AI) with your secure key, you’ll see an option for API keys on the dashboard.
- Click on the option “Get your API keys here” to access the [Anthropic API Keys](https://console.anthropic.com/settings/keys).

## Step 2: Create an Account or Log In

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant2.png"
/>

- Create a new account if you don’t already have one by providing necessary information like email, username, and password.
- If you already have an account, log in using your credentials.

## Step 3: Provide the Required Information

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant3.png"
/>

- Provide the necessary personal information to proceed with using Anthropic.
- Fill in the designated fields with the required information carefully and accurately.

## Step 4: Locate the "API Keys" Option and Generate a New Key

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant4.png"
/>

- Click on the "API Keys" option in the sidebar.
- After clicking, you'll see an option labeled "Create Key." Go ahead and click on it.

## Step 5: Copy and Secure the API Key

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant5.png"
/>

- Once generated, the new API key will be displayed.
- Copy the entire key string.
- **Important:** This key will only be shown once, so make sure to copy it.

## Step 6: Navigate to the Billing Section

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant6.png"
/>

- Click on the "Billing" option in the sidebar.
- Add a minimum of $5 to begin using Anthropic's services.

## Step 7: Complete Additional Details

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/ant7.png"
/>

- Here is some basic information that needs to be filled out before completing the billing process.
- **Note:** Adding funds is required to use the Anthropic model. The minimum amount you can add is $5.

## Step 8: Paste the API Key into the Designated Field

- Paste the API key into the designated field.
- You will now have full access to Anthropic models on RabbitHoles.AI.
