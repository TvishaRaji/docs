## Step 1: Open your RabbitHoles app and navigate to “AI Settings”

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters1.png"
/>
## Step 2: Click on xAI under Model settings

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters2.png"
/>
- To obtain your API key, navigate to the “xAI” model settings and click on **“Get your API key here”**.

## Step 3: Complete Sign-In Flow

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters3.png"
/>
- **Sign in** using your email and password associated with your xAI account if you already have an existing account.
- Alternatively, **authenticate** using **X (formerly Twitter)** or **Google**.
- If you do not have an account, click **"Sign Up"** to create one.

### Alternative (Sign Up Flow)

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters4.png"
/>
- **Enter your first name and last name.**
- **Set a password and confirm it.**
- Click **"Continue"** to proceed.

## Step 4: Go to API Console

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters5.png"
/>
- Click on **“Go to API Console”**.

## Step 5: Creating an API Key

- Go to **API Key** from the menu and click on **“Create API Key”**.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters6.png"
/>
- Choose a **name** for the API key.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters7.png"
/>
- Select the **endpoints** that you would like to configure it for.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters8.png"
/>
- Choose the **models** you want to configure the key for (by default, it is configured for all models).

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters9.png"
/>
- Once generated, the **API key will be displayed**.
- **Copy the entire key string**.
- **Important:** This key will only be shown once, so make sure to copy it immediately.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters10.png"
/>
## Step 6: Configure Model for xAI

- **Paste** the API key into the xAI placeholder in your **Model Settings** in RabbitHoles.
- Configure the **xAI model settings** by clicking on the **slider icon**.
- Add the correct **Model ID** for the model you would like to import.

### Model ID Reference from Google’s Official Documentation

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/twitters11.png"
/>

| **Model** | **Input** | **Output** |
| --- | --- | --- |
| grok-2-vision | TEXT, IMAGE | TEXT |
| grok-2-vision-latest | TEXT, IMAGE | TEXT |
| grok-2 | TEXT | TEXT |
| grok-2-latest | TEXT | TEXT |
| grok-vision-beta | TEXT, IMAGE | TEXT |
| grok-beta | TEXT | TEXT |

- **Set the Model ID** using the value provided above.
- **Assign a custom display name** for the model as needed.
- **Select the model type** from the dropdown menu, choosing between **"Chat"** and **"Reasoning"**.

You now have **full access to Grok models on RabbitHoles**!
