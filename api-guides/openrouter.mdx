### Step 1: Access the RabbitHoles Portal

<img style={{ borderRadius: '0.5rem' }} src="/images/openrouter1.png" />

After logging in to [RabbitHoles.ai](https://rabbitholes.ai) with your secure key, you'll see an option for API keys on the dashboard.
Click on the option **"Get your API keys here"** to access the OpenRouter API Keys.

### Step 2: Sign Up or Log In
To access the OpenRouter API, you need an account. If you don't already have one:
1. Visit the [OpenRouter website](https://openrouter.ai).
<img style={{ borderRadius: '0.5rem' }} src="/images/openrouter2.png" />
2. Click **Sign Up** and enter your details.
3. Verify your email and log in.

If you already have an account, simply **Log In** to proceed.

### Step 3: Visit the API Settings Page

<img style={{ borderRadius: '0.5rem' }} src="/images/openrouter3.png" />

Once logged in, navigate to the **OpenRouter API** page to manage your API credentials.

### Step 4: Generate an API Key
Once you have credits in your account:
1. Go to the **API Keys** section.
2. Click **Create new key**.
3. Copy and securely store your API key, as it won't be displayed again.

<img style={{ borderRadius: '0.5rem' }} src="/images/openrouter4.png" />

### Important Notes:
- The API key is a **long-lived access token**.
- It remains valid until manually revoked or deleted.
- **Keep your API key secure** and do not share it publicly.
