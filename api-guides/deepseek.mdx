### Step 1: Access the RabbitHoles Portal

<img style={{ borderRadius: '0.5rem' }} src="/images/deepseek1.png" />

After logging in to [RabbitHoles.AI](https://rabbitholes.ai) with your secure key, navigate to the **API Keys** section on the dashboard.
Click on **"Get your API keys here"** to access the Deepseek API Keys.

### Step 2: Sign Up or Log In to Deepseek
To use the Deepseek API, you need an account. If you don't have one:
1. Visit the [Deepseek website](https://deepseek.com).
2. Click **Sign Up** and fill in your details.
3. Verify your email and log in.

If you already have an account, simply **Log In** to proceed.

### Step 3: Navigate to API Settings

<img style={{ borderRadius: '0.5rem' }} src="/images/deepseek2.png" />

Once logged in, go to the **Deepseek API** settings page to manage your API credentials.

### Step 4: Add Credits
Before generating an API key, you need to add credits to your account:
1. Open the **Top Up** section.
2. Select the amount you want to add (**minimum recommended: $5**).
3. Complete the payment process.

### Step 5: Generate an API Key
Once your account has credits:
1. Navigate to the **API Keys** section.
2. Click **Create new key**.
3. Copy and store your API key securely, as it won't be displayed again.

<img style={{ borderRadius: '0.5rem' }} src="/images/deepseek3.png" />

<img style={{ borderRadius: '0.5rem' }} src="/images/deepseek3.png" />


### Important Notes:
- The API key is a **long-lived access token**.
- It remains valid until manually revoked or deleted.
- **Keep your API key secure** and do not share it publicly.
