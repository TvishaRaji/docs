## Step 2: Click on Google under Model settings

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/google2.png"
  />

To obtain your API key:
1. Navigate to the **Google** model settings.
2. Click on **"Get your API key here"**.

## Step 3: Click on "Create API key"

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/google3.png"
/>

- Click **"Create API key"**.
- Once generated, the API key will be displayed.
- Copy the **entire key string**.
- **Important:** The key will only be shown once, so copy it immediately.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/google4.png"
/>

## Step 4: Paste the API key in the Model Settings placeholder

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/google5.png"
/>

- A confirmation message **"Provider updated"** will appear upon successful update.
- **Note:** If an error message appears, verify that your API key is correct.
- Click **"Add new Model"**.

## Step 5: Configure model for Google
<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/google6.png"
/>

To configure a model:
- **Select the correct model ID** from the list below:

| Model Name | Input Type | Output Type | Description |
| --- | --- | --- | --- |
| [Gemini 2.0 Flash](https://ai.google.dev/gemini-api/docs/models/gemini#gemini-2.0-flash) `gemini-2.0-flash-exp` | Audio, images, videos, and text | Text, images (coming soon), audio (coming soon) | Next-generation multimodal model for diverse tasks |
| [Gemini 1.5 Flash](https://ai.google.dev/gemini-api/docs/models/gemini#gemini-1.5-flash) `gemini-1.5-flash` | Audio, images, videos, and text | Text | Fast and versatile |
| [Gemini 1.5 Flash-8B](https://ai.google.dev/gemini-api/docs/models/gemini#gemini-1.5-flash-8b) `gemini-1.5-flash-8b` | Audio, images, videos, and text | Text | High volume, lower intelligence tasks |
| [Gemini 1.5 Pro](https://ai.google.dev/gemini-api/docs/models/gemini#gemini-1.5-pro) `gemini-1.5-pro` | Audio, images, videos, and text | Text | Complex reasoning tasks |

- **Set the Model ID** using the value provided above.
- **Assign a custom display name** as needed.
- **Select the model type** from the dropdown menu: **"Chat"** or **"Reasoning"**.

You now have full access to Google models on RabbitHoles!
