## Step 1: Open [RabbitHoles.AI](http://RabbitHoles.AI)

![CleanShot 2024-10-23 at 13.03.05@2x.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/9a7e84bd-2b57-4d0d-b0b1-5ab16d1ada3b/CleanShot_2024-10-23_at_13.03.052x.png)

- After logging in to [RabbitHoles.AI](http://RabbitHoles.AI) with your secure key, you'll see an option for API keys on the dashboard.
- Click on the option "Get your API keys here" to access [OpenAI API Keys](https://platform.openai.com/docs/overview.API-keys).

## Step 2: Sign Up or Log In

![KEY 2.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/3ffea648-f9f9-44d3-aa23-e4bad322298d/KEY_2.png)

- If you don't have an account:
    - Click on the "Sign Up" button
    - Fill in the required information
    - Verify your email address
- If you have an account:
    - Click on the "Log In" button
    - Enter your credentials

## Step 3: Generate New API Key

![CleanShot 2024-10-23 at 09.16.44@2x.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/32cc5471-9cee-4602-afad-1d92c4a4817a/CleanShot_2024-10-23_at_09.16.442x.png)

- In the API dashboard, find the option to create a new API key
- It might be labeled as "Create new secret key" or similar
- Click on this option
- You may be asked to provide a name or description for the key (optional)
- Confirm the creation of the new API key

## Step 4: Copy and Secure the API Key

![KEY 4.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/519442c6-26aa-4f28-8401-8da1430928b0/KEY_4.png)

- Once generated, the new API key will be displayed
- Copy the entire key string
- **Important:** This key will only be shown once, so make sure to copy it immediately

## Step 5: Paste the API key into the designated field.

![CleanShot 2024-10-23 at 09.17.36@2x.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/1107bdd8-8ed5-4608-94b6-926eb16c05e1/CleanShot_2024-10-23_at_09.17.362x.png)

- Paste the API key into the designated field
- You will now have full access to OpenAI models on RabbitHoles
