## Step 1: Open your RabbitHoles app and navigate to “AI Settings”

![AI Settings](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/65d3ef39-536f-4dce-a9da-d1a19a4991a5/Screenshot_2025-01-16_at_9.50.00_PM.png)

## Step 2: Click on xAI under Model settings

![xAI Model Settings](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/f5805eef-b54d-4561-af44-8ed7903c19ed/Screenshot_2025-01-16_at_11.46.08_PM.png)

- To obtain your API key, navigate to the “xAI” model settings and click on **“Get your API key here”**.

## Step 3: Complete Sign-In Flow

![Sign In Flow](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/5d98e552-892a-4380-9867-5cf80a7dceb3/Screenshot_2025-01-16_at_11.46.57_PM.png)

- **Sign in** using your email and password associated with your xAI account if you already have an existing account.
- Alternatively, **authenticate** using **X (formerly Twitter)** or **Google**.
- If you do not have an account, click **"Sign Up"** to create one.

### Alternative (Sign Up Flow)

![Sign Up Flow](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/1ef9c9fc-c0b9-4fd2-874e-bda6a5f7941b/Screenshot_2025-01-16_at_11.49.37_PM.png)

- **Enter your first name and last name.**
- **Set a password and confirm it.**
- Click **"Continue"** to proceed.

## Step 4: Go to API Console

![API Console](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/a4d8a9c7-09b6-4077-9724-a9851a5e19f6/Screenshot_2025-01-16_at_11.52.45_PM.png)

- Click on **“Go to API Console”**.

## Step 5: Creating an API Key

- Go to **API Key** from the menu and click on **“Create API Key”**.

![Create API Key](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/673587ed-8d1b-4eef-82cb-a32be8412440/Screenshot_2025-01-16_at_11.54.21_PM.png)

- Choose a **name** for the API key.

![Name API Key](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/b10b199a-73bf-460d-a7ce-9518133c3243/1.png)

- Select the **endpoints** that you would like to configure it for.

![Select Endpoints](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/cab35460-812f-433f-ba7d-8b7d921a3dea/2.png)

- Choose the **models** you want to configure the key for (by default, it is configured for all models).

![Choose Models](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/241f5e51-f95c-4c6a-bc80-ac33cc3a8025/3.png)

- Once generated, the **API key will be displayed**.
- **Copy the entire key string**.
- **Important:** This key will only be shown once, so make sure to copy it immediately.

![API Key Generated](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/74ee9283-806d-4211-a0f9-ff951b65fd37/4.png)

## Step 6: Configure Model for xAI

- **Paste** the API key into the xAI placeholder in your **Model Settings** in RabbitHoles.
- Configure the **xAI model settings** by clicking on the **slider icon**.
- Add the correct **Model ID** for the model you would like to import.

### Model ID Reference from Google’s Official Documentation

![Model ID List](https://prod-files-secure.s3.us-west-2.amazonaws.com/953b4c50-29ab-4fca-aedf-c589f07d5485/d508d8d5-30bf-4feb-991d-67444e552db6/Screenshot_2025-01-17_at_12.05.22_AM.png)

| **Model** | **Input** | **Output** |
| --- | --- | --- |
| grok-2-vision | TEXT, IMAGE | TEXT |
| grok-2-vision-latest | TEXT, IMAGE | TEXT |
| grok-2 | TEXT | TEXT |
| grok-2-latest | TEXT | TEXT |
| grok-vision-beta | TEXT, IMAGE | TEXT |
| grok-beta | TEXT | TEXT |

- **Set the Model ID** using the value provided above.
- **Assign a custom display name** for the model as needed.
- **Select the model type** from the dropdown menu, choosing between **"Chat"** and **"Reasoning"**.

You now have **full access to Grok models on RabbitHoles**!
